Automated Graph generation:

MAIN Matlab scripts:
	- AutoTimePlot: plot the time usage for applications that have more than 4 
		records for all DB
	- AutoDataPlot: plot the data usage upload and download for all application 
		that have a data up/down usage

	NOTE: possible to change the parameters at the top of the script file


Other scripts:
	- are called from the main script

	- saveTightFigure: script downloaded on Matlab community website. Save the graph without large empty borders
